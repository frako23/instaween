# InstaWeen API

1. Clonar proyecto
2. `yarn install`
3. Clonar el archivo `.env.template` y renombrarlo a `.env`
4. Cambiar las variables de entorno
5. Levantar base de datos

```
docker-compose up -d
```

7. Levantar: `yarn: start:dev`
